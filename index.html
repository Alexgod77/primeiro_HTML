História dos computadores

Inicialmente a necessidade humana de realizar cálculos de forma mais rápida e precisa, levou a busca por máquinas que auxilia se nas operações simples.

Com o passar dos tempos foram desenvolvendo equipamentos que pudessem substituir o esforço humano em cálculos e problemas complexos, assim começaram a surgir os primeiros computadores.

Tendo vários inventores que contribuiram ao longo dos tempos no desenvolvimento de tecnologias, funções, tamanho e aplicações diferentes, sendo assim até os dias atuais.

A evolução da tecnologia resultou em máquinas que cabem na palma da mão, gerando aplicações em diversos setores da sociedade moderna.

O que são CLIENTs

É um termo usado em computação para representar um equipamento que consome os serviços de outro equipamento servidor, através do uso de uma rede de computadores numa arquitetura chamada cliente-servidor.

O que são SERVERs

É um equipamento ou software, com sistema de computação centralizada que fornece serviços a uma rede de computadores. Esses serviços podem ser de naturza distinta, por exemplo, arquivos, correio eletrônico, banco de dados.

Como é a estrutura básica de um HTML

É formada por apenas três tags HTML, HEAD, BODY.

HTML são as tags utilizadas apenas para guardar toda estrutura do site. Na tag HEAD é onde fica a estrutura comportamental do site, o conteúdo não aparece para o usuário, já na tag BODY é onde fica todo conteúdo do html que já aparece para o usuário, a estrutura física do site.

Lista Ordenada e Lista Não-Ordenada

São tags que servem para colocar uma ordem em alguma parte do texto, seja ela numerada ou não.

UL ordena a lista sem numeração dos itens

Exemplo

  • Azul
  • Verde
  • Amarelo

OL ordena a lista com numeração dos itens

Exemplo

  1. Azul
  2. Verde
  3. Amarelo

Há também a tag LI que fica dentro das tags OL e UL, que serve para listar os itens da lista.
